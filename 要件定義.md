
# フリーランスWeb制作者 プロフィールサイト 要件定義書

---

## 1. サイト概要

| 項目 | 内容 |
|---|---|
| サイト種別 | 個人プロフィールサイト（1ページスクロール型） |
| 制作者像 | データ分析の知見を持つ、エンジニア兼Web制作者 |
| サイトの目的 | スキル・制作イメージ・人柄を伝え、副業・制作依頼につなげる |
| ターゲット | Web制作を依頼したい個人事業主・小規模事業者、および活動に興味を持つ人 |

---

## 2. 誰の（ターゲット顧客像）

### メインターゲット
- 個人事業主・小規模事業者（飲食・サロン・ハンドメイド作家など）
- 「ホームページを作りたいが、大手制作会社は敷居が高い・費用が高い」と感じている人
- 制作者の人柄や雰囲気を見て依頼相手を決めたいタイプ

### サブターゲット
- 同じくWeb制作・デザインを学んでいる人（繋がり・参考にしたい）
- 制作者の活動・アウトプットに興味を持ってくれるフォロワー

---

## 3. 何を（顧客・訪問者の困りごと）

- どんな人が作っているのか、人柄・雰囲気が伝わらないと依頼しにくい
- 技術スペックだけ並べられても、自分のサイトの仕上がりをイメージしにくい
- 「完成品の実績がないと頼めないの？」という不安（＝制作イメージでカバー）
- 問い合わせのハードルが高い・何を聞いていいかわからない

---

## 4. どのように（このサイトで解決する）

| 課題 | 解決策 |
|---|---|
| 人柄が伝わらない | Aboutセクションでストーリー・価値観を丁寧に記述。猫モチーフで親しみやすさを演出 |
| 完成イメージが湧かない | 「制作イメージ」セクションで架空作品を複数掲載し、スタイルの幅を見せる |
| 技術が自分に合うか不安 | スキルをカード形式で視覚的に整理。学習中も正直に記載し誠実さを伝える |
| 問い合わせしにくい | シンプルなフォーム＋SNSリンクで複数の接触窓口を用意 |

---

## 5. サイト構成（セクション）

```
[Hero]
　キャッチコピー：「見やすさと使いやすさを大切にしたWeb制作。」
　サブコピー：シンプルで見やすいデザインを意識し、使いやすいWebサイトをお届けします
　ビジュアル：猫×PCのイラスト（オレンジ暖色背景）

[About]
　・現在の活動概要（Webデザイン・フロントエンド開発を学びながら制作）
　・Webデザインに興味を持った理由（データ分析 × 美しさと機能性の両立）
　・大切にしている考え方（シンプル・アクセシビリティ・ユーザー目線）

[Skills]
　HTML / CSS　／　JavaScript
　Tableau　　　／　SQL
　WordPress（学習中）　／　AI活用

[Works / 制作イメージ]
　・架空サイトデザイン（シンプルできれいなレイアウト）
　・バナー制作（視認性重視・わかりやすいバナー）
　・猫モチーフのデザイン（楽しみやすいデザイン表現）
　・シンプルなUIデザイン（ミニマルなインターフェース）

[Contact]
　SNSリンク：Twitter / Instagram / GitHub
　お問い合わせフォーム：お名前・メールアドレス・メッセージ・送信ボタン
```

---

## 6. デザイン方針

| 項目 | 内容 |
|---|---|
| 全体トーン | シンプル・落ち着き・やさしさ・信頼感 |
| アクセントカラー | テラコッタ・オレンジ系（#D45A2A 付近） |
| ベースカラー | オフホワイト・クリーム系 |
| モチーフ | 猫（Hero画像に黒猫＋PC）、ナチュラルな雰囲気 |
| レイアウト | 1カラム縦スクロール。Skillsは2列カード、Worksは2列グリッド |
| フォント方針 | 細め・読みやすい和文フォント。余白を広めに取る |
